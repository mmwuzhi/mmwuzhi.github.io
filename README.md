# 余白

Personal blog at [mmwuzhi.dev](https://mmwuzhi.dev) — Chinese / English / Japanese.

## Stack

- [Astro](https://astro.build) v6
- TypeScript
- Plain CSS (no framework)

## Development

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm build    # output → dist/
pnpm test     # unit tests
```
